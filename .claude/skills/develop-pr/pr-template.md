# PR テンプレート

draft PR の本文は以下の形式で作成すること。
言語やフォーマットの指定が CLAUDE.md の PR/Commit Conventions にある場合はそれに従う:

```markdown
## 概要

<!-- 変更の目的と概要を1-3文で -->

## 変更内容

<!-- 変更内容を箇条書きで。各項目は具体的に -->

-

## テスト計画

<!-- テストの実施内容 -->

- [ ] 既存テスト全パス (品質チェックコマンド)
- [ ] 新規テスト追加: (テストファイル名)
- [ ] 手動確認: (確認手順があれば)

## 備考

<!-- レビュアーに伝えたいことがあれば -->

---
Generated with [Claude Code](https://claude.ai/code)
```
